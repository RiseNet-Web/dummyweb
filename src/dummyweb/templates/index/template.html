Dummy is a MMORPG engine

The Dummy Project aims at building a minimalist,two-dimension, RPG Maker-like MMORPG Engine with its integrated map editor,servers,clients,and so on.

What is a MMORPG ?

Basically, this acronym stands for Massive Multiplayer Online Roleplaying Game. Quite popular nowadays, it is a roleplaying game when you can play with others by solving quests,donjons through your characters thanks to its abilities

Why DummyMeuporg ?

We love programming above everything (and we enjoy gaming, to be honest). We believe that we can build something valuable through involving as mush software developers / game addicts / news eager to learn as we can

How can I join the project ?

It is recommended to use Discord. We have a dedicated server for the project: discord because this is the place where we talk about the project,basically

The whole source are hosted on github : GitHub If you are experimented enough you can still take a look on the repos and try to get you hands dirty. If you don't know where to start get in touch with myself: @Ge0.
